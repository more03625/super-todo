# SuperToDo Phase 1 Specification

SuperToDo is a Personal Operating System — an analytics-first productivity platform.

## Phase 1 Modules

1. Authentication (JWT)
2. Dashboard
3. Task Categories
4. Tasks
5. Daily Score
6. Productivity Heatmap
7. Weekly / Monthly / Yearly Reports
8. Life Areas
9. Achievement System
10. Streak Dashboard
11. Charts

## Tech Stack

- **Backend:** Python 3.13, FastAPI, SQLAlchemy 2.0, Alembic, Pydantic v2, JWT, Passlib, APScheduler
- **Frontend:** Next.js 15, TypeScript, Bootstrap 5, Axios, React Query, Chart.js
- **Database:** PostgreSQL 16

See [AGENT.md](../AGENT.md) for coding standards.
